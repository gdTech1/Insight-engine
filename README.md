# Overview

Insight Engine is a data pipeline that processes raw data, transforms it using ETL, stores it in a database, and generates automated insights through a dashboard.

It simulates a real-world **data engineering + analytics system**.

---

# Features

- Data ingestion (CSV / API)
- ETL pipeline (extract, transform, load)
- PostgreSQL storage
- Automated insight generation
- Interactive Streamlit dashboard
